### Estructura de Datos
## Proposito
Este repositorio tiene el proposito de documentar ejercicios, deberes y materia acerca de la asignatura de Estructura de Datos de la Universidad Tecnica de Ambato,
de la carrera de Ingenieria de Software perteneciente al tercer semestre.

## Estructura del repositorio

El repositorio se divide principalemte en 2 parciales, cada parcial contiene carpetas donde se guardaran deberes, ejercicios y proyectos.
